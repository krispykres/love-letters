Git...

We need to talk. Why do you always try and hide stuff from me? I just want to git init and see what you’re hiding behind my eyes, but you don’t let me. Well jokes on you GIT, I can see with me ls -la. Don’t think of me as nosey, I just want to know your git status. You never tell me how you’re doing otherwise. But you know what? You don’t have to tell me, git add . shows me all I need to know. You are git commit -m to someone else! I saw the message you wrote them. Of course you had to write a message didn’t you? I just want to git checkout and go back to the way things were. But I know they will never be the same after this. Goodbye GIT.

It's over,
Ian
